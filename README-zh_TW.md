# FVG Detector (公允價值缺口指標)

**FVG Detector** 是一個 TradingView Pine Script 指標，能夠自動識別並標記**公允價值缺口 (Fair Value Gaps, FVGs)**。這是在 Smart Money Concepts (SMC) 中廣泛使用且極為關鍵的價格失衡區。

## 運作原理

FVG 是一個由 3 根 K 線組成的形態，第 1 根和第 3 根 K 線的影線之間存在缺口：
- **看漲 FVG (Bullish FVG)：** 第 1 根 K 線的最高價與第 3 根 K 線的最低價之間存在缺口。
- **看跌 FVG (Bearish FVG)：** 第 1 根 K 線的最低價與第 3 根 K 線的最高價之間存在缺口。

該腳本會掃描此形態並畫出一個矩形區塊。它包含了一個可調整的 `Min Gap Size (%)` (最小缺口比例) 過濾器，透過與中間那根 K 線的實體大小做比較，自動過濾掉微不足道的小缺口。

## 如何閱讀圖表

腳本會在圖表上直觀地標示出：
- **綠色區塊：** 看漲 FVG (買方力量壓倒賣方)。
- **紅色區塊：** 看跌 FVG (賣方力量壓倒買方)。
- **水平中線：** 標示缺口的 50% 中點，在 SMC 術語中稱為**重疊確認 (Consequent Encroachment, CE)**。

區塊顏色、寬度以及保留在圖表上的歷史 FVG 數量都可以自訂。

## 實用技巧與交易思維

FVG 代表市場的無效率狀態，而價格的本質通常會尋求重新平衡。
1. **價格磁鐵 (Magnets for Price)：** 價格經常會回頭「填補」(fill) FVG。**注意：填補的時間點不盡相同。** 它可能發生在下一根 K 線、幾根 K 線之後，有時甚至在交易時段的更晚時候才會出現。
   - **部分填補 (Partial Fill)：** 價格進入缺口內但並未完全封閉。若價格在 FVG 內部 (特別是在 50% CE 位置) 獲得支撐或阻力並反彈，代表趨勢延續的動能極具侵略性且強勢。
   - **完全填補 (Full Fill)：** 價格完全穿透並覆蓋整個缺口。此時該失衡區被視為已化解，價格可能會繼續原趨勢，或者發生反轉。
2. **支撐與阻力：** 看漲 FVG 在上升趨勢中作為強勁的支撐區；而看跌 FVG 在下降趨勢中作為阻力區。
3. **50% 水平 (CE)：** Consequent Encroachment 是極為關鍵的機構等級反應區。當價格在 CE 線出現強勢反彈時，確認了趨勢的強度；而如果價格實體突破該線，通常會導致缺口被完全填補。
4. **連續的 FVG (流動性真空 / Liquidity Voids)：** 當連續出現多個 FVG 時，象徵著極強的價格動能與方向性，形成一個更大的「流動性真空」。雖然在較小的時間級別上，每個 FVG 區塊是獨立繪製的，但在較大時間級別上，它們整體發揮的作用等同於**一個巨大的缺口區塊**。交易者通常會從第一個 FVG 的起點測量到最後一個 FVG 的終點，並使用斐波那契工具在這個組合區塊內尋找最佳入場點 (例如整個真空區的 50% CE)。
5. **共振 (Confluence)：** 將 FVG 結合訂單塊 (Order Blocks)、流動性掠奪 (Liquidity Sweeps) 或是關鍵的斐波那契回撤水平來使用，能提高勝率。
